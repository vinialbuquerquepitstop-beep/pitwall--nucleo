# Aprendizados

Memória evolutiva desta skill. Leia no início de cada dinâmica para aplicar o que já foi aprendido. Atualize no fim quando algo útil aparecer. É assim que a skill fica mais afiada com o tempo em vez de recomeçar do zero a cada conversa.

> Aviso honesto: este arquivo é uma convenção, não mágica. Ele só "aprende" se for de fato atualizado e lido. A memória do Claude por trás ajuda, mas este registro escrito é o que dá continuidade entre sessões e entre diferentes instâncias.

## Como registrar
Use entradas datadas, específicas e acionáveis. Formato:
`**[AAAA-MM-DD] — contexto** — Observação: ... ; Ação: ... ; Confiança: alta/média/baixa`

Mantenha seções abaixo. Periodicamente, consolide entradas repetidas e resolva contradições explicitamente (não deixe duas regras opostas convivendo).
